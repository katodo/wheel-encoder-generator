Exporting images

You can export your wheel encoder to an image file, too. One never knows when that might come in handy.

Once you've designed an encoder, export it to a PNG image by selecting Export from the File menu or click the Export tool in the toolbar. A file chooser dialog will appear. Select the directory/folder where you want to store the image, and type in the name ending with .PNG.

Notes

The current version of WheelEncoderGenerator exports only to a PNG file and the image generated is always 600 by 600 pixels. Future releases will provide options to configure the size and image format.
