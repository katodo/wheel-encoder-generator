Binary Code encoders

Binary code encoders are absolute encoders that encode the standard binary numeral system on the disc.

The binary number sequence, such as 000, 001, 010, 011, 100, 101, 110, 111, is encoded directly on the encoder disc. Sequential positions may differ by more than one bit and so this coding style may introduce errors in position, but it is the most convenient from a software standpoint. Simply read the sensors as bits making up a word and that word is the integer value of the position. No translation necessary. You can compare position values directly.
