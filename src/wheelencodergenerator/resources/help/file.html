File Menu
New Creates a new encoder
Open Opens a previously saved encoder
Save Saves the current encoder configuration. If the file hasn't been saved before, this menu selection behaves the same as Save As
Save As Saves the encoder with a name you provide using a file chooser dialog box.
Export Creates an image file with a name you provide using a file chooser dialog box
Print Prints the encoder to scale using a print dialog box
Exit Quits the JDE.  Doesn't appear on Mac OS X systems. Instead, select Quit from the WheelEncoderGenerator menu.
