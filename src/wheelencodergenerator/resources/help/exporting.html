Exporting images

You can export your wheel encoder to an image file, too. One never knows when that might come in handy.

Once you've designed an encoder, export it to a PNG, GIF, or JPEG image by selecting Export from the File menu or click the Export tool in the toolbar.

First a dialog will appear asking you for the resolution (size) of the image in pixels and the file format to use. The resolution you enter will be used to set the width and height of the encoder image. Select the file format as either GIF, JPEG, or PNG.

Next, a file chooser dialog will appear. Select the directory/folder where you want to store the image and type in the filename.

Notes

On Windows, you don't need to enter a file extension. If you enter a file extension it needs to match the file format you selected in the export dialog.

On Mac OS X, it's best if you simply enter the filename without an extension. If you do enter an extension, it needs to match the file format you previously selected. Use .png if you selected the PNG file format, .jpg for JPEG, and .gif for GIF.
