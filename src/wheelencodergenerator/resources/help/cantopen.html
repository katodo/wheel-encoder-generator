If you can't open an encoder

Here are some troubleshooting tips if you can't open an encoder file.

Be sure you didn't accidentally select the wrong file.

Make sure you have read permissions to the file you're trying to open. Try opening it with a text editor (WordPad on Windows or TextEdit on Mac).

Is the file really a WheelEncoderGenerator (.weg) file? Is the file corrupt? Open it with a text editor. It should look something like this:

#Wheel Encoder Settings
#Fri Oct 15 21:33:08 MDT 2010
encoder.innerDiameter=5
encoder.quadratureTrack=false
encoder.outerDiameter=50
encoder.resolution=10
encoder.type=0
encoder.indexTrack=false
encoder.numbering=0
encoder.inverse=false
