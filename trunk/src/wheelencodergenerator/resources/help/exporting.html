Exporting images

You can export your wheel encoder to an image file, too. One never knows when that might come in handy.

Once you've designed an encoder, export it to a PNG image by selecting Export from the File menu or click the Export tool in the toolbar.

First a dialog will appear asking you for the size of the image in pixels. The value you enter will be used to set the width and height of the wheel image.

Next, a file chooser dialog will appear. Select the directory/folder where you want to store the image, and type in the name ending with .PNG.

Notes

The current version of WheelEncoderGenerator exports only to a PNG file. Future releases will provide options to select other image formats as well as Postscript and PDF.
